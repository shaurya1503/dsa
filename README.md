# DSA Practice : Striver's A2Z Sheet

My solutions to Data Structures & Algorithms problems, following Striver's A2Z DSA Sheet, in C++.

## Structure

Each problem gets its own folder containing:
- 'Problem_statement' : the question
- 'Brute_force.cpp' : first working approach
- 'Better_solution.cpp' : improved approach (if applicable)
- 'Optimal_solution.cpp' : most efficient solution
